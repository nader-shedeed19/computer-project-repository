Types of Programming language:

1-Machine and assembly languages .
2-Algorithmic languages like FORTRAN,ALGOL,LISB and C.
3- Business-oriented languages like COBOL and SQL.
4-Education-oriented languages like BASIC,Bascal,Logo, Hypertalk,object- oriented languages like C++,Ada,Java and Visual Basic
5-Declarative languages like PROLOG,LISB and ML
6- Scripting languages like PERL and UNIX
7-Document formatting languages like Tex, PostScript and SGML.
8-Wold Wide Web display languages like HTML,XML and Web scripting.
